Python字符串常用操作

2017-12-10

下面操作所使用的name的值如下

大小写相关

capitalize()

这一行首字母转换为大写

title()

所有单词的首字母都大写

casefold()

全部转换为小写

lower()

全部转换为小写

upper()

全部转换为大写

swapcase()

把大写转小写，小写转大写

istitle()

判断首字母是否全部为大写

isupper()

判断是否全部大写

查找相关

find()

查找字符串中包含的指定字符，并返回字符串中最左边的指定字符的下标

rfind()

从左往右数，找到最右边那个值的下标

判断相关

isalnum()

判断字符串中是不是同时包含字母和数字，如果同时包含了字母和数字，而且没有包含其它空格和任何特殊字符那么就返回True

isalpha()

判断字符串中是不是只有英文字母，如果只有英文字母而其没有其它数字、空格和任何特殊字符，就返回True

isdecimal()

判断是否为十进制

isdigit()

判断是否为整数

isnumeric()

判断是否只包含数字

isspace()

判断字符串是否为空格

endswith()

判断字符串是否以指定字符结尾，如果是，就返回True

startswith()

判断字符串是否以指定字符开始，如果是，就返回True

打印格式相关

format()

join()

ljust()

打印100个字符，不够的话右边的全部用指定字符来填补，这里用*

rjust()

打印100个字符，不够的话左边的全部用指定字符来填补，这里用*

center()

替换相关

expandtabs()

把字符串中的tab转换成多个空格，这里转换成了30个空格

replace()

替换字符串中的指定字符，可以指定替换次数

其他常用操作

split()

把字符串按照指定字符分成一个列表，默认以空格分割成一个列表

splitlines()

按照换行符，把字符串分割成一个列表

lstrip()

去除左边的换行

​

rstrip()

去除右边的换行

strip()

去除两边的换行

count()

encode()

打印python目前的编码模式
